# 🚀 AI Semantic Search using Endee

## 📌 Overview
This project is a semantic search system built using the Endee vector database.

It allows users to input a query and retrieves the most relevant result using vector similarity.

## 💡 Features
- Uses Endee as vector database
- Converts text into embeddings
- Performs semantic search

## 🧠 Tech Stack
- Python
- Endee
- Sentence Transformers

## ⚙️ Architecture
User Query → Embedding → Endee Search → Result

## ▶️ Setup Instructions
pip install -r requirements.txt  
python data.py
